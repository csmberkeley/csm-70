A joint density function for two random variables X and Y is a non-negative function f: $\textbf{R}^2$ $\rightarrow$ \textbf{R} that has total integral equal to 1 ($\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x,y)dxdy = 1$). \\
Joint distribution of X and Y:
\[\P[a \leq X \leq b, c \leq Y \leq d] = \int_{c}^{d} \int_{a}^{b} f(x,y)dxdy \].
