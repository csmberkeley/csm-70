\question Show that $E(aX + b) = aE[X] + b$ where $X$ is any random variable. 

\begin{solution}[1 cm]
First, for the expectation:
$$\E[aX + b] = \sum_{x \in X} (aX + b) P(X = x)= a\sum_{x \in X} X \cdot P(X = x) + \sum_{x \in X} b P(X = x)$$$$= a\E[X] + b\sum_{x \in X} P(X = x) = a\E[X] + b$$.
\end{solution}
