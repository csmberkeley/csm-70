\question Show that $\var(X) = \E((X - \mu)^2) = \E(X^2) - \mu^2$ where $X$ is any random variable and $\mu = \E[X]$. 

\begin{solution}[1.5 cm]
\begin{center}
$\var(X) = \E((X - \mu^2) = \E[X^2 - 2\mu X + \mu^2]$ \\[2mm]
$= \E[X^2] - 2\mu\E[X] + \mu^2 $ (due to linearity of expectation) \\[2mm]
$ = \E[X^2] - \mu^2 $
\end{center}
\end{solution}
